# IBM Data Analyst Capstone Project

## Technology Trends Analysis using the Stack Overflow Developer Survey

This project was completed as part of the **IBM Data Analyst Professional Certificate**.

The objective of this project was to analyze the Stack Overflow Developer Survey dataset to identify technology trends among software developers and communicate findings using dashboards and visualizations.

---

## Project Objectives

- Collect and prepare survey data
- Clean and transform the dataset
- Perform Exploratory Data Analysis (EDA)
- Build interactive dashboards
- Generate business insights

---

## Dataset

**Source**

Stack Overflow Developer Survey

## Dashboard Visualizations

As part of the capstone project, I designed interactive dashboards using **Google Looker Studio** to analyze the Stack Overflow Developer Survey dataset.

The dashboards include:

- Technology trends developers want to work with
- Technology currently used by developers
- Programming language trends
- Database trends
- Cloud platform adoption
- Developer demographics

📊 **Dashboard PDF:** [Google Looker Studio Dashboards](dashboards/Google_Looker_Studio_Dashboards.pdf)

## Key Insights

- Identified the most commonly used programming languages.
- Analyzed trends in databases and cloud platforms.
- Explored developer demographics from the survey.
- Presented findings through dashboards and visualizations.
