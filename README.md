# IBM Data Analyst Capstone Project

## Technology Trends Analysis using the Stack Overflow Developer Survey

This project was completed as part of the **IBM Data Analyst Professional Certificate**.

The objective of this capstone project was to analyze the **Stack Overflow Developer Survey** dataset to identify technology trends among software developers and communicate insights through interactive dashboards and visualizations.

---

## Project Overview

This project follows the complete data analytics workflow:

- Data Collection
- Data Cleaning
- Data Wrangling
- Exploratory Data Analysis (EDA)
- Data Visualization
- Dashboard Development
- Insight Generation

---

## Project Objectives

- Collect and prepare survey data
- Clean and transform the dataset
- Perform Exploratory Data Analysis (EDA)
- Analyze technology trends
- Build interactive dashboards
- Generate actionable insights

---

## Dataset

**Source:** Stack Overflow Developer Survey

The dataset contains information about developers worldwide, including:

- Programming Languages
- Databases
- Cloud Platforms
- Web Frameworks
- Education
- Age
- Professional Experience

---

- Microsoft Excel

## Dashboard Visualizations

Interactive dashboards were created using **Google Looker Studio** to analyze technology trends among developers.

The dashboards include:

- Technology developers want to work with
- Technology developers currently work with
- Programming language trends
- Database trends
- Cloud platform adoption
- Developer demographics

## Key Insights

Some of the insights obtained from the analysis include:

- Popular programming languages among developers
- Most widely used databases
- Adoption of major cloud platforms
- Technology preferences for future work
- Distribution of developers by age and education
